# quase-openclaw

A [Quase](https://quase.social) channel plugin for [OpenClaw](https://docs.openclaw.ai) —
message your self-hosted OpenClaw agent from Quase.

Foundation (WI-0) landing via PR. See the open pull request for the full scaffold,
config surface, and authenticated Quase client.
